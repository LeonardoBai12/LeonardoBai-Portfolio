About

I'm an Android Engineer with over 5 years of professional experience designing and shipping mobile applications used in production at scale. My work spans payments, security SDKs, and cross-platform mobile development.

I specialize in Kotlin, Jetpack Compose, Clean Architecture, and MVVM. I care deeply about testability, code quality, and building architectures that teams can actually maintain. I've introduced testing cultures from scratch, led SDK redesigns, and contributed CI/CD pipelines with Docker and GitHub Actions.

Beyond Android, I build with Kotlin Multiplatform (KMP/Compose Multiplatform) targeting both Android and iOS, write Swift and SwiftUI for native iOS work, use Ktor for backend services, and occasionally Flutter.
